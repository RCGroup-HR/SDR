┌───────────────────────────────────────────────────────────────┐
│  ⚡ PASO 1: INSTALAR BASE DE DATOS (Solo 1 vez)              │
└───────────────────────────────────────────────────────────────┘

OPCIÓN A (Más Fácil):
1. Haz click derecho en: instalar-carnets.bat
2. "Ejecutar como administrador"
3. Espera a que termine

OPCIÓN B (PowerShell):
1. Click derecho en: instalar-carnets.ps1
2. "Ejecutar con PowerShell"

OPCIÓN C (MySQL Workbench):
1. Abre MySQL Workbench
2. Abre archivo: backend\sql\INSTALAR_SISTEMA_CARNETS.sql
3. Ejecuta todo (Ctrl+Shift+Enter)


┌───────────────────────────────────────────────────────────────┐
│  🌐 PASO 2: ACCEDER AL SISTEMA                               │
└───────────────────────────────────────────────────────────────┘

1. Abre tu navegador

2. Ve a: http://localhost:5174

3. Inicia sesión con tu usuario

4. En el menú lateral:
   📋 Mantenimientos
      └── 🎴 Gestión de Carnets  ← ¡CLICK AQUÍ!


┌───────────────────────────────────────────────────────────────┐
│  ⚠️ PROBLEMAS COMUNES                                        │
└───────────────────────────────────────────────────────────────┘

ERROR: "No veo el menú 'Gestión de Carnets'"
SOLUCIÓN:
  ✓ Refresca la página (F5)
  ✓ Verifica que tengas permisos para "carnet_federacion"

ERROR: "Error 404" o "Error al cargar carnets"
SOLUCIÓN:
  ✓ Ejecuta el PASO 1 (instalar-carnets.bat)
  ✓ Verifica que el backend esté corriendo en puerto 3000

ERROR: "Error 500"
SOLUCIÓN:
  ✓ Ejecuta el script SQL (PASO 1)
  ✓ Verifica que la base de datos sea: sdr_domino
