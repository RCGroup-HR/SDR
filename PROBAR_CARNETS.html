🔐 Autenticación

Ingresa tu token de autenticación (obtenlo iniciando sesión en la aplicación)

El token debe tener el formato: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

📋 Antes de Comenzar

  • Asegúrate de haber ejecutado: instalar-carnets.bat
  • El servidor debe estar corriendo en: http://localhost:3000
  • Necesitas un token válido (inicia sesión primero)
  • Los directorios se crean automáticamente al subir archivos
